## Como rodar localmente

1. Clone o repositório

   ```bash
   git clone https://github.com/Malvino30-06/portfolio-malvino.git

   ```

2. Entre na pasta

   ```cd
   portfolio-malvino

   ```

3. Instale as dependências

   ```npm
   install

   ```

4. Inicie o servidor de desenvolvimento
   ```npm
   run dev
   ```

Acesse em http://localhost:5173 _(ou a porta que aparecer)_
